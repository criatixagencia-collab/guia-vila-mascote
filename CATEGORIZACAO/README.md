# Categorizacao - Guia Vila Mascote

Pasta para revisar a organizacao dos estabelecimentos coletados no scrap de 1 ano da @vila.mascote.

Arquivos principais:

- `taxonomia_categorias_v1.md`: definicao das categorias e regra de multiplas categorias por estabelecimento.
- `estabelecimentos_categorizados_v1.md`: listagem em Markdown, agrupada por categoria principal.
- `estabelecimentos_categorizados_v1.csv`: mesma listagem em formato de planilha, melhor para filtro e importacao futura.

Resumo da V1:

- 147 estabelecimentos/itens analisados.
- 126 marcados como `Listar`.
- 8 marcados como `Revisar`.
- 13 marcados como `Nao listar`, por parecerem posts editoriais, eventos sem estabelecimento claro, duplicidades ou remocao solicitada.

Observacao: esta categorizacao e a base de curadoria usada para gerar o `dados.js` do site.
